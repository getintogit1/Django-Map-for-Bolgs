## ğŸ“¦ Installation

1. Clone the repository:
    ```bash
    git clone <repo-url>
    cd project-folder
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Apply migrations:
    ```bash
    python manage.py migrate
    ```

4. Run the development server:
    ```bash
    python manage.py runserver
    ```

5. Visit: [http://127.0.0.1:8000](http://127.0.0.1:8000)
